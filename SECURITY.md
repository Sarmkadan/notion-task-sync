# Security Policy

## Supported Versions

Currently, the following versions are supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of Notion Task Sync seriously. If you discover a security vulnerability, please do NOT open a public issue.

Instead, please report it via one of the following methods:

1. **GitHub Private Vulnerability Reporting** (Preferred)
   Use GitHub's built-in private advisory feature by visiting the "Security" tab in this repository and submitting a new advisory.

2. **Email**
   Contact us directly at: **rutova2@gmail.com**

### Acknowledgment

We will acknowledge receipt of your vulnerability report within **48 hours**. We will then work with you to understand the issue, provide a fix, and responsibly disclose the vulnerability once it is resolved.
